# React Ecommerce

* [Material UI](https://material-ui.com/)

* [Redux](https://react-redux.js.org/)

### [Live Demo](https://pizza-fire-ecommerce.netlify.app/)

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jawariaalvi32/React-ecommerce
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Run
 ```sh
   npm start
   ```
